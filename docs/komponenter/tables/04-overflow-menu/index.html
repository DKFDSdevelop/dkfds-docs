Overflow menu

Overflow menu is a separate component that can be also used in other places of the website. It is a button that expands a menu with some options.

<div class="overflow-menu">
    <button class="overflow-menu-btn">

        <i class="icon icon-dots-vertical "></i>

    </button>
    <div class="overflow-menu-options position-right">
        <button>Option 1</button>
        <button>Option 2</button>
        <button>Option 3</button>
        <button>Option 4 is longer text</button>
    </div>
</div>

SCSS classes

SCSS class Description
overflow-menu The container for all overflow-menu items.
overflow-menu-btn The button that expands the menu.
overflow-menu-options The container for menu options.
