Sådan (do)

Sådan: Reducér antallet af handlingstrin ved filoverførsel så meget som muligt. Det er en kompliceret opgave, som vil være svær for en del computerbrugere. Mange har svært ved at håndtere opgaver, der omfatter mange handlingstrin og flere brugergrænseflader (OECD, 2016; Nielsen, 2016).

Ikke sådan (don't)

Ikke sådan: Undgå unødvendige knapper og dermed handlingstrin. Hvis der både er en "Gennemse"-knap og en "Vedhæft"-knap, vil nogle brugere ikke vide, hvad de skal gøre og måske begå fejl.
